# Predicting Crime Rate

This project focuses on predicting crime rates using demographic data from the United States.  
The analysis is implemented in a Jupyter Notebook and includes data exploration, preprocessing, model development, and evaluation.

## Project Structure

- `pro.ipynb` – main notebook containing the analysis and prediction model
- `pro_dokumentacja.pdf` – detailed project documentation
- `LICENSE` – MIT license

## Goal

The goal of the project is to analyze demographic variables and use them to build a model capable of estimating crime rates.

## How to Use

Open the `pro.ipynb` notebook in Github, Jupyter Notebook, JupyterLab, or Google Colab and run the cells step by step.
